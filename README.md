READ ME
-------
Programming Fundamentals Assignment 1of2 -
Inventory management system making use of multiple programming paradigms with a GUI for user interaction and csv files to record all changes to data.
This system has buttons to Add, Remove, altter the quantity and clear all items in a list.
Each item has its own ID, name and quantity.
I used tkinter for the GUI as it is a comprehensive library to aid in the development of the project.
-------------------------------------------------------------------------------------------------------
Software requirements - 
Python 3.14.2
Source code located in this repository
To run, simply press the run button
